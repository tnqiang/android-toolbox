# 隐私声明 / Privacy Notice

最后更新：2025

## 简要说明

**android-toolbox 是一款完全本地运行的工具，不会上传任何用户数据到任何服务器。**

## 应用收集与使用的数据

为实现核心功能，本应用会通过 ADB 协议在你的电脑与已连接的 Android 设备之间读取以下信息，**全部仅在本机使用，绝不上传**：

- 设备序列号 / 型号 / 品牌 / Android 版本 / SDK
- 设备 IMEI、MEID（仅在你查看「设备详情」时通过 ADB 即时读取）
- 已安装应用的包名、版本号、安装路径、图标与名称
- 设备的截图、相册图片（仅在你主动使用截图 / 相册功能时）

## 数据存储位置

- 应用名称 / 图标缓存：`%APPDATA%\android-toolbox\`（Windows，0.5.4 起；早期版本曾为 `%APPDATA%\手机助手\`）
- 用户偏好设置：同上
- 卸载应用时，可选择是否保留以上缓存

## 网络访问

本应用本身**不主动访问任何网络**。仅以下情况会产生网络流量：
- Electron 框架自身的诊断（默认禁用）
- 若未来加入「检查更新」功能，将仅访问 GitHub Releases 接口，且可在设置中关闭
